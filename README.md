# ✨ Overview

**AnalyticaAI** is an AI-integrated analytics platform that transforms CSV datasets into interactive dashboards, intelligent visualizations, AI-generated insights, anomaly detection reports, and downloadable analytics reports—all within an elegant Streamlit interface.

Whether you're a student, analyst, researcher, or business professional, AnalyticaAI enables data-driven decisions without requiring advanced programming knowledge.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/praneethavaranasi/AnalyticaAI.git

cd AnalyticaAI
```

---

## Create Virtual Environment

```bash
python -m venv .venv
```

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements-app.txt
```

---

## Run Application

```bash
streamlit run App.py
```

Application starts at

```
http://localhost:8501
```
